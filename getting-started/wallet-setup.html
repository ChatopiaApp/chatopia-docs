Wallet Setup

To interact with Chatopia and manage your AI character NFTs, you need a Web3 wallet. This guide covers setting up MetaMask and preparing for platform integration.

Installing MetaMaskâ€‹

Desktop Setupâ€‹

  1. Visit metamask.io and verify URL authenticity
  2. Download browser extension for Chrome, Brave, Opera, or your preferred supported browser
  3. Install from official browser store and pin to toolbar
  4. Click MetaMask icon and select "Create a new wallet"
  5. Create strong password and accept terms of service

Mobile Setupâ€‹

  1. Download MetaMask from iOS App Store or Google Play Store
  2. Verify app authenticity before installation
  3. Open app and tap "Create a new wallet"
  4. Follow setup instructions and create secure password

Wallet Security Fundamentalsâ€‹

Seed Phrase Protectionâ€‹

Critical Security Rules:

  • Never share your seed phrase with anyone under any circumstances
  • Never store seed phrase digitally, in photos, or on devices
  • Chatopia team will never ask for your seed phrase or private keys
  • Seed phrase provides complete access to all wallet funds and NFTs

User Responsibility:

  • Secure backup and storage methods are your responsibility
  • Wallet recovery procedures are handled by your wallet provider
  • Contact MetaMask support for wallet-specific recovery assistance
  • Research and implement security practices appropriate for your situation

Additional Security Measuresâ€‹

  • Use unique, strong passwords for all wallet access
  • Keep device software and browsers updated with latest security patches
  • Consider hardware wallets for significant holdings and long-term storage
  • Use separate devices or browsers for high-value transactions

Network Configurationâ€‹

Blockchain Networkâ€‹

Current Status:

  • Specific network will be announced before Q3 2025 testnet launch
  • Common options include Ethereum, Polygon, Binance Smart Chain and other EVM-compatible chains
  • Network addition instructions will be provided when details are available

Preparation:

  • Learn how to add custom networks to MetaMask through official documentation
  • Understand gas fee concepts for different blockchain networks
  • Familiarize yourself with network switching procedures in wallet interface

Transaction Costsâ€‹

Gas Fee Requirements:

  • All blockchain interactions require native network tokens for transaction fees
  • Costs vary significantly based on network congestion and transaction complexity
  • Fees apply to character minting, training, breeding, and marketplace activities
  • Users responsible for monitoring network status and managing fee costs

Platform Connectionâ€‹

Wallet Integrationâ€‹

  1. Ensure MetaMask is properly installed and wallet created
  2. Visit official Chatopia platform when launched
  3. Click "Connect Wallet" in platform interface
  4. Select MetaMask and approve connection request
  5. Verify wallet address appears correctly in platform interface

Connection Securityâ€‹

  • Always verify platform URL matches official Chatopia domains
  • Never connect wallet to suspicious or unverified websites
  • Review all connection permissions before approval
  • Disconnect wallet when finished using platform

Token Requirementsâ€‹

Native Network Tokensâ€‹

  • Required for all blockchain transaction fees and smart contract interactions
  • Amount needed varies by network selection and current gas price conditions
  • Available from centralized exchanges and decentralized trading platforms
  • Users responsible for acquiring appropriate amounts before platform use

CHATOPIA Tokensâ€‹

  • Required for character training operations and breeding functionality
  • Available through Q4 2025 IDO and subsequent exchange listings
  • Users determine their own token acquisition needs based on intended platform usage
  • Always verify official token contract addresses before purchasing

Alternative Wallet Optionsâ€‹

Desktop Walletsâ€‹

Popular Alternatives:

  • Coinbase Wallet (beginner-friendly interface with educational resources)
  • Trust Wallet (mobile-first wallet with desktop browser support)
  • Rainbow Wallet (Ethereum-focused with intuitive design)

Hardware Walletsâ€‹

Enhanced Security Options:

  • Ledger (industry-standard hardware security for cold storage)
  • Trezor (open-source hardware wallet with strong security features)
  • Hardware wallets can connect through MetaMask for platform interaction

Mobile Integrationâ€‹

  • WalletConnect protocol enables mobile wallet connections to desktop platforms
  • Scan QR codes to securely connect mobile wallets to browser interfaces
  • Maintains mobile wallet security while accessing desktop platform features

Troubleshooting Platform Connectionâ€‹

Common Connection Issuesâ€‹

Wallet Not Connecting:

  • Refresh browser page and attempt connection again
  • Ensure wallet is unlocked and active in browser
  • Verify correct blockchain network is selected in wallet
  • Clear browser cache and restart browser if connection fails

Transaction Problems:

  • Verify sufficient native tokens available for gas fees
  • Check blockchain network status for congestion or maintenance
  • Review transaction details for accuracy before confirmation
  • Contact platform support for persistent technical issues

Platform-Specific Supportâ€‹

  • Platform connection issues should be reported to Chatopia support
  • Wallet functionality problems should be directed to wallet provider support
  • Blockchain network issues are outside platform and wallet provider control
  • Community forums available for user-to-user assistance and troubleshooting

Security Best Practicesâ€‹

Daily Security Habitsâ€‹

  • Lock wallet when not actively using Web3 applications
  • Verify website URLs carefully before connecting wallet
  • Review all transaction details thoroughly before approval
  • Keep wallet software updated with latest versions

Threat Preventionâ€‹

  • Bookmark official Chatopia platform domains for direct access
  • Be cautious of phishing websites and fake customer support
  • Report suspicious activity to community moderators immediately
  • Start with small test transactions when using new features

User Responsibilityâ€‹

Users are fully responsible for:

  • Wallet security and seed phrase protection
  • Transaction decisions and financial risk management
  • Researching and implementing appropriate security measures
  • Understanding blockchain technology risks and limitations
